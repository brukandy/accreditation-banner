CODICE PER ELEMENTOR - BANNER ACCREDITAMENTI
=============================================

Copia e incolla in un widget HTML di Elementor (posizionalo sopra la navbar):

<style>
/* Reset padding Elementor */
.elementor-widget-html {
    margin: 0 !important;
    padding: 0 !important;
}
.elementor-widget-container {
    margin: 0 !important;
    padding: 0 !important;
}

/* Iframe responsive wrapper */
.accreditation-banner-wrapper {
    position: relative;
    width: 100%;
    height: 50px;
    overflow: hidden;
}

@media (max-width: 768px) {
    .accreditation-banner-wrapper {
        height: 45px !important;
    }
}

@media (max-width: 480px) {
    .accreditation-banner-wrapper {
        height: 40px !important;
    }
}
</style>

<div class="accreditation-banner-wrapper">
    <iframe 
        src="https://brukandy.github.io/accreditation-banner/" 
        width="100%" 
        height="100%" 
        frameborder="0"
        scrolling="no"
        style="border: none; display: block; margin: 0; padding: 0; overflow: hidden;">
    </iframe>
</div>

<script>
// Auto-resize iframe
window.addEventListener('message', function(e) {
    if (e.data.accreditationHeight) {
        const iframe = document.querySelector('iframe[src*="accreditation-banner"]');
        if (iframe) {
            iframe.style.height = e.data.accreditationHeight + 'px';
        }
    }
});
</script>
